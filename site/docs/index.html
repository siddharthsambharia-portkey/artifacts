Quick reference

The most common patterns:

<!-- 1. Drop into any .html file deployed on Artifact --> <script src="/artifact.js"></script> <script> await artifact.ready(); // Identity console.log(artifact.me.email); // "alice@corp.com" // Database (create + realtime subscribe) const posts = artifact.db.collection('posts'); posts.subscribe({ onCreate: (doc) => renderPost(doc) }); await posts.create({ title: 'hello', body: 'world' }); // AI (streaming) const s = await artifact.ai.chat([{ role:'user', content:'Hi' }], { stream: true }); for await (const chunk of s) output.textContent += chunk; // File upload const f = await artifact.files.upload(fileInput.files[0]); img.src = f.url; </script>

Examples

Five example apps ship in the examples/ folder of the repo:

  • live-poll — Create and vote in real-time polls; realtime DB subscribe
  • multiplayer-cursors — See teammates' cursors; WebSocket presence
  • lunch-vote — Thumbs up/down food options; KV store + governance
  • team-dashboard — Display team metrics; warehouse query + AI summary
  • guestbook — Sign the company guestbook; identity + Slack notify

Deploy any example: cd examples/live-poll && artifact deploy
